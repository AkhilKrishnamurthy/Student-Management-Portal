This is a project of a student management portal. It shows how ReactJS, NodeJS and MySQL database together.

Follow the steps to run the application:

Create a MySQL DB table with name 'login'.
add fields 'username' and 'password' and give the desired username and password.

open a command prompt inside backEnd and run "node index.js"
open another command prompt inside frontEnd and run "npm start" which will open the browser and the application would run.

Important:
once the browser is opened, run http://localhost:3000/login.
Login with the username and password you provided in MySQL 'login' table and add students details.
